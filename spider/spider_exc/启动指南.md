# 快速启动指南

## 前置条件

1. **Java 17+** 已安装
2. **Maven 3.6+** 已安装
3. **MySQL 8.0+** 已安装并运行
4. **前端项目 spider_web** 已启动（运行在 localhost:3000）

## 快速开始

### 1. 创建数据库

```sql
CREATE DATABASE spider_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. 执行SQL脚本

```bash
# Windows PowerShell
mysql -u root -p spider_db < docs\sql\schema.sql

# Linux/Mac
mysql -u root -p spider_db < docs/sql/schema.sql
```

### 3. 配置数据库连接

编辑 `src/main/resources/application.yml`，修改数据库用户名和密码：

```yaml
spring:
  datasource:
    username: root
    password: 你的密码
```

### 4. 启动前端项目

在另一个终端窗口：

```bash
cd ../spider_web
npm install
npm run dev
```

确保前端运行在 `http://localhost:3000`

### 5. 安装 Playwright 浏览器

```bash
# 方式1：运行测试时会自动安装
mvn test-compile

# 方式2：手动安装（如果方式1失败）
# 下载 Playwright 并安装浏览器
```

### 6. 运行测试

#### 方式1：使用IDE（推荐）

1. 打开项目
2. 找到 `FourStepIndicatorTest` 类
3. 依次运行以下测试方法（按顺序）：
   - `testCollect()` - 采集数据
   - `testParse()` - 解析数据
   - `testProcess()` - 处理数据
   - `testComplete()` - 持久化数据

#### 方式2：使用Maven命令

```bash
# 运行所有测试（按顺序）
mvn test

# 或者分别运行每个测试
mvn test -Dtest=FourStepIndicatorTest#testCollect
mvn test -Dtest=FourStepIndicatorTest#testParse
mvn test -Dtest=FourStepIndicatorTest#testProcess
mvn test -Dtest=FourStepIndicatorTest#testComplete
```

## 常见问题排查

### 问题1：数据库连接失败

**错误信息**：
```
Communications link failure
```

**解决方法**：
1. 检查MySQL服务是否运行
2. 检查 `application.yml` 中的数据库配置
3. 确认数据库用户有足够权限

### 问题2：前端页面无法访问

**错误信息**：
```
net::ERR_CONNECTION_REFUSED
```

**解决方法**：
1. 确认前端项目已启动：`cd ../spider_web && npm run dev`
2. 检查前端是否运行在 `http://localhost:3000`
3. 在浏览器中手动访问前端页面验证

### 问题3：Playwright浏览器未安装

**错误信息**：
```
BrowserType.launch: Executable doesn't exist
```

**解决方法**：
```bash
# 重新安装浏览器
mvn clean test-compile

# 或者手动安装
java -cp target/classes com.microsoft.playwright.CLI install chromium
```

### 问题4：元素定位失败

**错误信息**：
```
Element not found: #invoice-sign-0
```

**解决方法**：
1. 确认前端页面已加载完成
2. 检查前端页面的元素ID是否正确
3. 增加等待时间：`page.waitForTimeout(3000)`
4. 使用浏览器开发者工具检查实际元素ID

### 问题5：测试顺序错误

**错误信息**：
```
采集记录不能为空
```

**解决方法**：
必须按照以下顺序运行测试：
1. testCollect()
2. testParse()
3. testProcess()
4. testComplete()
