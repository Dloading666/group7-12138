# XX电子税务平台 - 模拟网站

这是一个模拟重庆市政府税局网站的前端项目，用于RPA爬虫测试。

## 功能说明

本项目包含以下页面，展示指标计算所需的所有参数：

1. **首页/登录页** - 模拟税局网站入口
2. **企业信息查询页面** - 显示纳税人识别号(taxNo)、统一社会信用代码(uscCode)
3. **发票查询页面** - 显示发票列表，包含：
   - sign（发票类型：销项/进项）
   - state（发票状态：正常/异常）
   - invoice_time（发票开票时间）
   - jshj（价税合计）
4. **申请页面** - 显示申请日期(appDate)

## 技术栈

- Vue 3
- Vite
- Vue Router

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```
